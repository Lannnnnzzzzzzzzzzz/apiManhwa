Manhwaindo Scraper API

Fast and reliable API for scraping manhwa data from manhwaindo.app. Access series information, chapters, and images.

API Endpoints

GET /api

Get API information and available endpoints

// Response
{
  "success": true,
  "message": "Manhwaindo API",
  "version": "1.0.0",
  "endpoints": { ... }
}
GET /api/project?page=1

Get project updates from homepage

Query Parameters:

  • page - Page number (default: 1)
// Response
{
  "success": true,
  "page": 1,
  "data": [...]
}
GET /api/lastupdate?page=1

Get latest updates

Query Parameters:

  • page - Page number (default: 1)
// Response
{
  "success": true,
  "page": 1,
  "data": [...]
}
GET /api/popular

Get popular manhwa series

// Response
{
  "success": true,
  "data": [
    {
      "title": "Solo Leveling",
      "slug": "solo-leveling",
      "rating": "9.5"
    }
  ]
}
GET /api/genres

Get all available genres

// Response
{
  "success": true,
  "totalGenres": 42,
  "data": [
    {
      "id": "action",
      "name": "Action"
    }
  ]
}
GET /api/series-list?page=1

Get all series with filters

Query Parameters:

  • page - Page number (default: 1)
  • order - update | popular | latest | title
  • type - manhwa | manhua | manga
  • status - ongoing | completed | hiatus
// Example: /api/series-list?page=1&order=update&type=manhwa

// Response
{
  "success": true,
  "page": 1,
  "filters": {
    "order": "update",
    "type": "manhwa"
  },
  "data": [...]
}
GET /api/series/:slug

Get detailed information about a series

// Example: /api/series/solo-leveling

// Response
{
  "success": true,
  "data": {
    "title": "Solo Leveling",
    "slug": "solo-leveling",
    "status": "Completed",
    "author": "Chugong",
    "genres": ["Action", "Fantasy"],
    "chapters": [...]
  }
}
GET /api/chapter/:slug

Get all images for a specific chapter

// Example: /api/chapter/solo-leveling-chapter-1

// Response
{
  "success": true,
  "data": {
    "title": "Solo Leveling Chapter 1",
    "slug": "solo-leveling-chapter-1",
    "images": [
      "https://example.com/image1.jpg",
      "https://example.com/image2.jpg"
    ],
    "prevChapter": null,
    "nextChapter": "solo-leveling-chapter-2"
  }
}
GET /api/search?q=query

Search for manhwa by title

Query Parameters:

  • q - Search query (required)
// Example: /api/search?q=solo

// Response
{
  "success": true,
  "query": "solo",
  "data": [
    {
      "title": "Solo Leveling",
      "slug": "solo-leveling"
    }
  ]
}
GET /api/download/:slug

Get download information for a chapter

// Example: /api/download/solo-leveling-chapter-1

// Response
{
  "success": true,
  "data": {
    "title": "Solo Leveling Chapter 1",
    "images": [
      {
        "url": "https://...",
        "filename": "001.jpg"
      }
    ]
  }
}
POST /api/download-multiple

Download multiple chapters as ZIP

// Request Body
{
  "slugs": [
    "solo-leveling-chapter-1",
    "solo-leveling-chapter-2"
  ]
}

// Response: ZIP file download
GET /api/download-pdf/:slug

Download chapter as PDF

// Example: /api/download-pdf/solo-leveling-chapter-1

// Response: PDF file download
